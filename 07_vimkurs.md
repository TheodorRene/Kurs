



# Konfigurering 
- RC-fil - Run configuration 
- Alt legges i .vimrc
- Klarer seg lenge uten å endre så mye på den

# Mac og linux
- Lag en fil som heter ".vimrc" i hjemmappa di

# Windows
- Det er knot, men google det så ordner det seg

# Remapping
- imap jk <Esc>
- nmap <C-p> !python % <CR>
- nmap <Space>s :vsplit . <CR>
- nmap <C-s> :w <CR>
